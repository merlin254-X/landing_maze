About the Project

This maze project was inspired by a desire to create a visually engaging game while learning advanced concepts like raycasting and 3D rendering. This maze project aims to create an engaging 3D game using raycasting and real-time rendering. It features player movement with collision detection, smooth controls for walking and rotating, and efficient rendering for optimal performance. The focus is on delivering an immersive, fluid maze-solving experience. The development timeline spanned several weeks of work and study. It serves as a Holberton School Portfolio project.

Developed by Derick Marine.
